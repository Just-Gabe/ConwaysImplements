# ConwaysImplements

Varias implementações do jogo da vida de Conway em diversas linguagens.

## Objetivo

O objetivo desse repo é juntar meus estudos em diferentes linguagens em um só lugar, o objetivo de cada linguagem é fazer de uma forma ou outra o jogo da vida de Conway.
Pode ser usando só o terminal, pode ser usando interfaces gráficas, páginas web, o importante é aprofundar os estudos de cada linguagem para implementar o algoritmo e aprender coisas novas.
